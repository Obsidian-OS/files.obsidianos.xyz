Installation Options

Option 1:

Pre-built ISO

The easiest way to get started. Download a pre-built ISO and follow our installation guide.

Editions are listed above.

Option 2:

Build Your Own

For advanced users who want to customize their installation. Create your own ISO from scratch.

Build from Source

Option 3:

Arch-based Installer

Install ObsidianOS directly from an existing Arch Linux system using our installer script.

sudo bash -c "$(curl -fsSL https://arbs.obsidianos.xyz)"

Option 4:

Manual install from an Arch-based host

Install arch-install-scripts and obsidianctl to do an manual install.

Get obsidianctl
